# not-your-chat-app
A simple chat built using websockets

## Requirements

As a user I want to enter a chat room
As a user I want to send a message
As a user I want to recieve a message
As a user I want to leave a chat room

### Stretch
As a user I want to be able to view messages pass messages in a chat room
As a user I want to login
As a user I want to be able to clear my history
As a user I want to remain anonymous
As a user I don't want my data stored on a server
As a user I want to be able to see all my data
